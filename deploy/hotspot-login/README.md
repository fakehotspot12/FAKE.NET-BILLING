# Integrasi QR Login Hotspot

Pemindai QR berjalan pada `https://voucher.fakehotspot.net/scan-voucher.html`
agar browser dapat membuka kamera secara langsung. Router tidak memerlukan input
file, `jsQR.js`, atau skrip pemindai lokal.

Contoh netral tersedia pada `login-example.html` dan `login-example.css`. Contoh
tersebut tidak memuat logo, harga, alamat, atau data Site milik pengembang.

Tambahkan kontrol berikut di dalam form `name="login"`, tepat di sebelah tombol
submit. Ikon mengarah ke halaman pemindai kamera portal voucher.

```html
<div class="login-actions">
  <input type="submit" value="masuk" class="btn login-submit">
  <button class="qr-login-button"
          type="button"
          onclick="openVoucherScanner()"
          title="Scan QR voucher"
          aria-label="Scan QR voucher">[ikon QR]</button>
</div>
```

Fungsi tombol membawa URL login lokal sebagai `return_url`. Scanner tetap dapat
melanjutkan login meskipun QR lama hanya berisi username/password.

```html
<script>
function openVoucherScanner() {
  var base = 'https://voucher.fakehotspot.net/scan-voucher.html';
  var target = base + '?return_url=' + encodeURIComponent(window.location.href.split('#')[0]);
  var popup = window.open(target, '_blank');
  if (!popup) window.location.href = target;
}
</script>
```

Contoh ukuran kontrol:

```css
.login-actions {
  display: grid;
  grid-template-columns: minmax(0, 1fr) 2.25rem;
  gap: .5rem;
}

.qr-login-button {
  display: grid;
  place-items: center;
  width: 2.25rem;
  min-height: 2.25rem;
}
```

QR dari billing membuka URL login Site dengan fragment `fnb_autologin`. Agar
autentikasi CHAP dan PAP berjalan otomatis, tambahkan bridge berikut setelah
form login:

```html
<script>
(function voucherAutoLogin() {
  if (!window.location.hash) return;
  var params = new URLSearchParams(window.location.hash.slice(1));
  if (params.get('fnb_autologin') !== '1') return;
  var username = params.get('username') || '';
  var password = params.get('password') || username;
  var loginForm = document.forms.login;
  if (!username || !loginForm) return;
  loginForm.username.value = username;
  loginForm.password.value = password;
  try {
    window.history.replaceState(null, document.title, window.location.pathname + window.location.search);
  } catch (error) {}
  window.setTimeout(function () {
    if (typeof window.doLogin === 'function') window.doLogin();
    else loginForm.submit();
  }, 250);
}());
</script>
```

## Izin kamera

Halaman login Hotspot biasanya dibuka oleh mini-browser captive portal. Sebagian
Android/iPhone tidak memberikan izin kamera pada mini-browser tersebut.

1. Tekan ikon QR di sebelah tombol Masuk.
2. Jika permintaan kamera muncul, pilih **Izinkan**.
3. Jika kamera dibatasi captive portal, tekan **Buka di Browser**.
4. Di Chrome: tekan ikon pengaturan situs, pilih Izin, lalu aktifkan Kamera.
5. Di iPhone: buka halaman pada Safari, lalu pilih Izinkan saat diminta.

Scanner selalu melakukan decode di browser. Data voucher tidak dikirim ke server
scanner. URL fragment yang berisi kredensial juga tidak ikut dikirim dalam request
HTTP.

## Walled garden

Domain scanner harus dapat dibuka sebelum pelanggan login. Terapkan rule berikut
dan izinkan IP hasil resolusi DNS port 443. Contoh idempoten tersedia pada
`voucher-scanner-walled-garden.rsc`.

```routeros
/ip hotspot walled-garden add dst-host=*voucher.fakehotspot.net dst-port=443 action=allow comment="Generated by Billing - Voucher Scanner"
```

URL login tiap NAS diisi melalui `Monitoring > Site > Edit > URL Login Hotspot`.
Jangan menyimpan satu URL router global pada Pengaturan umum.
